# superhero-app

Meal planner where you can browse recipes to add them to a weekly calendar
This project is simple app that is fetching data from Food Recepies Api.  
Data is locally stored inside LocalStorage.

---

## Tchnologies
- Vue 3
- Vue-Router
- Vuetifyjs
- Vite
- Pinia

```

```
